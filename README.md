# ART | HOUSE — Sitio web

Sitio oficial de **ART | HOUSE**, productora audiovisual en la Ciudad de México.

- Sitio estático de un solo archivo: `index.html` (no requiere build).
- Bilingüe ES/EN, con portafolio de reels, AI Studio, eventos y formulario de contacto.
- Publicado en **Netlify** con deploy automático desde este repositorio.

## Editar el sitio
1. Abre `index.html` y haz tus cambios (o edítalo directo en GitHub con el ícono del lápiz ✏️).
2. Guarda / haz *commit*.
3. Netlify publica los cambios solo en ~1 minuto.

Guías incluidas:
- `GUIA-DESPLIEGUE.md` — publicar y editar (general).
- `GUIA-AUTODEPLOY-Y-DOMINIO.md` — auto-deploy con GitHub + conectar el dominio arthouse.mx.
